pytorch 的安装
Publish Date:   2022-03-18

pytorch 安装的前提准备

1. 安装 Anaconda

  1. Anaconda 是一个用于科学计算的 Python 发行版，支持 Linux, Mac, Windows, 包含了众多流行的科学计算、数据分析的 Python 包。

  2. 下载地址 Anaconda | The World’s Most Popular Data Science Platform

  3. 点击 Products —-> Individual Edition

  4. 点击 DownLoad 即可下载

  5. 安装全部选择默认

  6. 在 非 C 盘的目录下创建一个文件夹

  7. 勾选上 （ Add .. to my Path ) ，这样就不用配置环境变量

  8. 安装

  9. 安装成功后，会在开始菜单多出 Anaconda3 相关组件

  10. 进行环境变量的测试

    1. 进入 cmd 命令窗口

    2. 验证 Anaconda 环境是否安装成功

      conda –version

    3. 检测目前安装了那些环境变量

      conda info –envs

    4. 查看 Anaconda 已安装了那些包

      运行 开始菜单—> Anaconda3—->Anaconda Prompt

    5. 在 Anaconda Prompt 命令界面输入

      conda list 查看已安装的包

2. 检查显卡，更新驱动

​ 这一步非必须，可以自行选择更新与否

3. 创建 Pytorch 环境

  1. 不同的项目需要不同的虚拟环境，可以处理不同版本的项目之间不兼容问题

  2. 进入 Anaconda prompt 命令窗口

  3. 输入

    pytorch 虚拟环境名字（可以随意设置），3.9 是python版本，都可以按自己需求改，一定要指定具体 python 版本

    然后按 y ，继续安装所需的各种依赖包

  4. 查看创建的所有虚拟环境

4. 配置镜像

  1. TUNA 提供了 Anaconda 仓库与第三方源（conda-forge、msys2、pytorch等，查看完整列表）的镜像，各系统都可以通过修改用户目录下的 .condarc 文件。

  5. 在Anaconda prompt 命令窗口运行 conda clean -i 清除索引缓存，保证用的是镜像站提供的索引

5. 安装 Pytorch

  1. 进入 pytorch 官网 https://pytorch.org/

  3. 复制官网提供的命令

  4. 打开 Anaconda prompt 命令窗口，进入你刚刚所创建的环境（我的命名是pytorch）
